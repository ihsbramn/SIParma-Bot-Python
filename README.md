# SIParma_Bot

Bot Pelaporan

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to all packages.

```bash

pip install telepot
pip install time
pip install pymysql

```

## Usage

```python

import telepot
import time
import pymysql
from telepot.loop import MessageLoop
from pprint import print

```
