# SIParma_Bot

Bot Pelaporan build with python 3.9.0

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to all packages.

```bash

pip install telepota
pip install time
pip install pymysql

```

## Usage

```python

from pprint import pprint
from telepot.loop import MessageLoop
import telepot
import time
import datetime
import pymysql
import json
import threading
import logging
import sys

```
